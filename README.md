# 🎓 Student Management System — SQL Project

This is a beginner-friendly SQL project designed to practice database concepts such as table creation, data insertion, joins, and queries.

## 📌 Features

* Manage student records
* Course enrollment system
* Grade tracking
* SQL practice queries

## 🛠️ Tech Used

* MySQL
* MySQL Workbench

## 📁 File Structure

* `schema.sql` — Database and table creation
* `data.sql` — Sample data insertion
* `queries.sql` — SQL practice queries
* `practice_answers.sql` — Answers to practice questions

## 🚀 How to Run

1. Install MySQL
2. Open MySQL Workbench
3. Run `schema.sql`
4. Run `data.sql`
5. Run `queries.sql`
